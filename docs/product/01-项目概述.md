# Mole 项目概述

## 项目简介

Mole 是一款专为 macOS 设计的系统清理和优化工具，集成了 CleanMyMac、AppCleaner、DaisyDisk 和 iStat Menus 的核心功能于单一二进制文件中。

## 核心价值

- **一体化工具包**：多个专业工具的功能整合
- **深度清理**：扫描并删除缓存、日志和浏览器残留，回收数十GB空间
- **智能卸载**：彻底删除应用及其启动代理、偏好设置和隐藏残留
- **磁盘洞察**：可视化使用情况，管理大文件，重建缓存和刷新系统服务
- **实时监控**：CPU、GPU、内存、磁盘和网络的实时统计，诊断性能问题

## 目标用户

1. **开发者**：需要清理开发工具缓存（Xcode、npm、Docker等）
2. **专业用户**：需要深度系统维护和性能优化
3. **普通用户**：希望简单高效地清理Mac空间
4. **系统管理员**：需要批量管理和优化多台Mac

## 产品定位

- **免费开源**：MIT许可证
- **命令行优先**：专业用户友好
- **安全可靠**：内置严格保护机制
- **高性能**：Go + Shell混合架构
- **可扩展**：支持白名单和自定义配置

## 核心功能模块

### 1. Clean（深度清理）
回收系统空间，删除不必要的缓存和临时文件

### 2. Uninstall（智能卸载）
完整删除应用及其所有关联文件

### 3. Optimize（系统优化）
重建系统数据库、刷新服务、安全加固

### 4. Analyze（磁盘分析）
交互式可视化磁盘使用情况

### 5. Status（系统监控）
实时系统健康仪表板

### 6. Purge（项目清理）
清理开发项目的构建产物

### 7. Installer（安装包清理）
查找并删除散落的安装程序文件

## 技术特点

- **混合架构**：Shell脚本（灵活性）+ Go语言（性能）
- **安全设计**：路径验证、白名单保护、操作日志
- **并发处理**：充分利用多核CPU
- **缓存优化**：智能缓存减少重复扫描
- **TUI界面**：基于Bubble Tea的现代终端UI

## 使用场景

1. **定期维护**：每月运行一次深度清理
2. **空间不足**：快速回收数十GB空间
3. **应用卸载**：彻底删除不需要的应用
4. **性能诊断**：实时监控系统资源
5. **开发清理**：清理项目构建缓存
6. **系统优化**：重建索引和刷新服务

## 安全保障

- ✓ 路径验证：防止删除系统文件
- ✓ 符号链接检查：防止跟随恶意链接
- ✓ 应用保护：关键应用白名单
- ✓ 干运行模式：预览操作
- ✓ 操作日志：完整审计跟踪
- ✓ 超时保护：防止无限等待
